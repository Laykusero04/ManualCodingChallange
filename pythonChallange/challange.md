1. Sum of Two Numbers

Task: Write a function that returns the sum of two numbers.
Example:

Input: 3, 5

Output: 8

2. FizzBuzz

Task: Print numbers from 1 to n. For multiples of 3, print "Fizz". For multiples of 5, print "Buzz". For numbers that are multiples of both 3 and 5, print "FizzBuzz".
Example:

Input: 15

Output:
1
2
Fizz
4
Buzz
Fizz
7
8
Fizz
Buzz
11
Fizz
13
14
FizzBuzz

3. Palindrome Check

Task: Write a function to check if a string is a palindrome (reads the same forward and backward).
Example:

Input: "racecar"

Output: True

Input: "hello"

Output: False

4. Fibonacci Number

Task: Return the n-th Fibonacci number.
Example:

Input: 5

Output: 5

5. Factorial

Task: Write a function that returns the factorial of a number n (i.e., n!).
Example:

Input: 4

Output: 24

6. Count Vowels

Task: Count the number of vowels in a given string.
Example:

Input: "hello world"

Output: 3

7. Reverse Words

Task: Reverse the order of words in a sentence.
Example:

Input: "hello world"

Output: "world hello"

8. Find Maximum

Task: Find the maximum number in a list.
Example:

Input: [1, 2, 3, 0, -1]

Output: 3

9. Prime Check

Task: Write a function to determine if a number n is prime.
Example:

Input: 11

Output: True

Input: 4

Output: False

10. Merge Sorted Lists

Task: Merge two sorted lists into one sorted list.
Example:

Input: [1, 3, 5], [2, 4, 6]

Output: [1, 2, 3, 4, 5, 6]
